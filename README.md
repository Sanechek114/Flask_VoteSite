# ТЕХНИЧЕСКОЕ ЗАДАНИЕ (ПРЕДВАРИТЕЛЬНОЕ)
 Веб-сервис для создания и проведения голосований  



## 1. Цель и общая идея
Создать простой веб-сайт, на котором пользователи смогут регистрироваться, создавать опросы, участвовать в них и просматривать результаты. 



## 2. Ожидаемый результат
- Рабочий сайт, доступный в браузере
- Возможность зарегистрироваться, войти и выйти
- Создание опросов с выбором вариантов ответов
- Голосование с защитой от повторного участия в одном опросе
- Наглядное отображение итогов
- Личный раздел с историей созданных опросов



## 3. Основные блоки функционала
| Блок | Что предполагается |
|------|-------------------|
| **Пользователи** | Регистрация, вход, выход, базовый профиль |
| **Опросы** | Создание, просмотр списка, переход к деталям |
| **Голосование** | Выбор варианта, фиксация результата, блокировка повторных голосов |
| **Результаты** | Подсчёт, расчёт процентов, визуальное отображение |
| **Управление** | Просмотр своих опросов, базовые действия (удаление/редактирование) |
